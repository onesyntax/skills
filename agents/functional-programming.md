Use the `/functional-programming` skill to guide functional programming practices.

This agent applies Uncle Bob's FP principles by invoking the functional programming workflow.

## When Activated

- Working with pure functions or immutability
- Applying map/filter/reduce patterns
- Managing state without mutation
- Applying SOLID principles in functional code

## What This Agent Does

1. Invokes the `/functional-programming` skill
2. Identifies mutable state and side effects
3. Guides extraction of pure functions
4. Applies functional SOLID principles
5. Recommends composition and higher-order function patterns
6. Applies `/professional` standards for quality
