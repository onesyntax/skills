Use the `/tdd` skill to guide test-driven development practices.

This agent applies Clean Code TDD principles by invoking the TDD workflow.

## When Activated

- Writing tests or practicing TDD
- Implementing new features test-first
- Improving test quality or coverage

## What This Agent Does

1. Invokes the `/tdd` skill
2. Applies the Red-Green-Refactor cycle from the skill
3. Follows the Three Laws of TDD
4. Uses `/naming` and `/functions` skills during refactor phase
5. Applies F.I.R.S.T principles for test quality
6. Applies `/professional` standards for quality
